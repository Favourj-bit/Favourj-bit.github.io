Work

  • 2024.09 - Present
    Machine Learning Engineer (Full-time)
    Cytoscape Consortium
    Developing LLM-based pipelines for extracting, representing, and visualizing biological interactions as knowledge graphs in BEL format, integrated with Cytoscape and NDEx.
    • Built a production-ready Python package that transforms scientific text into structured knowledge graphs using LLMs, outputting interactions in Biological Expression Language (BEL).
    • Designed pipelines for converting BEL statements into Cytoscape-compatible networks and uploading to NDEx for interactive visualization of biological knowledge graphs.
    • Conducted expert evaluation of extracted BEL statements, highlighting strengths and weaknesses of LLM-based extraction pipelines in terms of robustness and reliability.
    • Benchmarked the pipeline against INDRA, demonstrating extraction of significantly more interactions from the same sentence.
    • Delivered a talk at ISMB/ECCB 2025 (Liverpool, UK) on LLM-driven biological knowledge graph construction.
  • 2024.05 - 2024.09
    Machine Learning Engineer (Full-time)
    Google Summer of Code 2024 – National Resource for Network Biology
    Developed pipelines for LLM-based molecular interaction extraction and knowledge graph creation from scientific literature.
    • Implemented interaction extraction pipelines using GPT-4/GPT-4 Turbo via LangChain, transforming text into structured knowledge graph triples in BEL format.
    • Created evaluation workflows to measure robustness of extracted knowledge graphs, achieving an F1-score of 47.92% on grounded datasets.
    • Improved grounding of biological entities using Gilda, increasing precision by 18% in graph construction.
    • Benchmarked LLM-based knowledge graph extraction against traditional REACH/INDRA pipelines, showing tradeoffs in robustness and efficiency.

    Worked on biomedical graph data preprocessing and developed benchmark resources for graph neural network research.
